# OWLI 

___
Questa parte del progetto permette di fare una prima chiamata api, tramite fetch, per trovare i risultati dei libri corrispondenti alla categoria inserita nell'apposita textbox, e un'eventuale seconda per visualizzarne la descrizione 

## Realizzazione
___
L'applicazione è stata realizzata con HTML, CSS e Javascript. 
___
✅ Funzionalità Principali

Ricerca di libri tramite interfaccia utente

Fetch dinamico per ottenere titoli di libri da un’API

Visualizzazione dei risultati in pagina
